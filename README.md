## 项目简介

这是一个由兰大学生自发制作的**纯静态新生入学手册网页**，旨在帮助新生快速了解校园生活的方方面面，替代分散在微信群、公众号、贴吧里的零散信息。

### 核心特色

- **一站式入口**：宿舍、食堂、地图、校园卡、军训、转专业、校历、快递、通勤等高频问题集中展示
- **稳定维护**：采用「稳定说明 + 当年官方通知链接」的方式，避免内容过时
- **公开访问**：纯静态页面，可部署到 GitHub Pages / Vercel / 学校服务器，新生无需登录即可查看
- **响应式设计**：完美适配桌面、平板、手机等各种设备
- **互动体验**：实时搜索、分类筛选、留言墙词云、目录导航

## 本地开发

### 环境要求

无需特殊环境，任意浏览器即可打开。推荐使用本地服务器预览：

```bash
# 方式一：Python
cd 项目目录
python -m http.server 8080

# 方式二：Node.js
npx serve .

# 方式三：直接打开
双击 index.html
```

然后访问 http://localhost:8080

## 维护手册的四条原则

宿舍分配、军训安排、快递站点、校历和转专业政策都会随年份变化。这个网页适合采用「稳定说明 + 当年通知链接」的维护方式。

1. **发布前核对** — 当年迎新通知、教务处通知和学院群公告
2. **标注更新时间** — 把容易变化的时间、地点、电话标注更新时间
3. **收集反馈补充** — 收集新生反馈，把高频问题补成独立卡片
4. **部署上线** — 部署到 GitHub Pages、Vercel、Netlify 或学校服务器

## 部署

### GitHub Pages（已配置）

1. 推送代码到 GitHub
2. 在仓库 Settings → Pages 中选择 `master` 分支
3. 等待 1-2 分钟即可访问

### Vercel（已配置）

1. Vercel 已连接 GitHub 仓库，每次 `git push` 自动部署
2. Framework Preset 选 **Other**（纯静态，无构建步骤）
3. 也可点击顶部「Deploy with Vercel」按钮一键部署
