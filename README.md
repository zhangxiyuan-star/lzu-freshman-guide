# 兰州大学新生入学手册

面向兰州大学新生的公开网页手册，把报到前后最容易卡住的事整理在一个网页里。

## 维护手册的四条原则

宿舍分配、军训安排、快递站点、校历和转专业政策都会随年份变化。这个网页适合采用「稳定说明 + 当年通知链接」的维护方式。

1. **发布前核对**当年迎新通知、教务处通知和学院群公告。
2. **标注更新时间**：把容易变化的时间、地点、电话标注更新时间。
3. **收集反馈补充**：收集新生反馈，把高频问题补成独立卡片。
4. **部署上线**：部署到 GitHub Pages、Vercel、Netlify 或学校服务器，使所有人可访问。

## 内容板块

- **入学事项全览**：宿舍、食堂、校园卡、军训、转专业、校历、快递、通勤等 12 个指南条目
- **两校区 · 一张图**：榆中校区、城关校区、通勤校车介绍
- **资源链接集**：13 个官方入口链接

## 技术说明

- 纯静态 HTML + CSS + JavaScript
- 响应式设计，适配桌面和移动端
- 支持搜索、筛选、目录导航
