Maintain

更新原则

宿舍分配、军训安排、快递站点、校历和转专业政策都会随年份变化。这个网页适合采用“稳定说明 + 当年通知链接”的维护方式。

  1. 发布前核对当年迎新通知、教务处通知和学院群公告。
  2. 把容易变化的时间、地点、电话标注更新时间。
  3. 收集新生反馈，把高频问题补成独立卡片。
  4. 部署到 GitHub Pages、Vercel、Netlify 或学校服务器，使所有人可访问。

非官方整理页面。重要事项以兰州大学及各学院当年正式通知为准。
